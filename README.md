# PinkyLam-blog
基于node webkit 开发的桌面笔记管理应用。

## 项目涉及
- nodeJs
- nwjs
- MDUI(前端框架)
- layui
- editor(markdown插件)
- redis

## PinkyLam-blog 使用教程
[使用教程](https://github.com/handexing/PinkyLam-blog/raw/master/doc/INSTALL_DOC.md)
